# UAS_algoritma
Program mencakup implementasi materi pemrograman lanjutan untuk mengelola dan mengolah data nilai mahasiswa menggunakan bahasa C secara terpadu. Penyimpanan Data: Menggunakan array satu dimensi untuk NIM dan matriks (array dua dimensi) untuk data nilai. Manajemen Data (CRUD): Dapat menampilkan, menambah, mengubah, dan menghapus data mahasiswa. Pengolahan Statistik: Menghitung rata-rata kelas serta mencari nilai tertinggi dan terendah. Fungsi Rekursif: Menghitung faktorial dan deret Fibonacci berdasarkan jumlah mahasiswa.
Cara Menjalankan Program :
Pastikan Anda memiliki compiler C (seperti GCC) atau menggunakan IDE Code::Blocks.
Compile file: gcc program.c -o program
Jalankan program: ./program
